# 🕹️ HTML5 기술 보고서: GAME UNIVERSE 엔진 분석

> **본 문서는 GAME UNIVERSE 플랫폼과 Infinity Blocks 게임의 설계 아키텍처 및 핵심 알고리즘에 대한 기술적 분석을 담고 있습니다.**

---

## 🏗️ 플랫폼 아키텍처 (Platform Architecture)

### 1. 실시간 데이터 동기화 및 썸네일 프로세싱
플랫폼 코어(`platform.js`)는 Firebase Firestore와 Storage를 결합하여 데이터의 실시간성과 영속성을 보장합니다.
- **이미지 자동 최적화**: 외부 이미지 URL 사용 시 발생하는 보안 정책(CORS) 문제를 해결하기 위해, 브라우저 내 **Canvas API**를 활용합니다. 이미지를 캔버스에 그려 리사이징한 후 JPEG 압축(0.7 품질) 및 Base64 인코딩을 거쳐 저장함으로써 스토리지 효율을 극대화합니다.
- **실시간 데이터 구독**: `onSnapshot` 리스너를 통해 데이터베이스의 변경 사항을 실시간으로 감지합니다. 비동기적으로 게임 라이브러리를 업데이트하여 모든 사용자가 최신 상태의 게임 목록을 볼 수 있도록 설계되었습니다.

## 🎮 핵심 게임 엔진 분석: Infinity Blocks

### 1. 절차적 계단 생성 알고리즘 (Procedural Stair Generation)
무한히 이어지는 게임 세계를 구현하기 위해 **결정론적 랜덤(Deterministic Random)** 로직을 기반으로 계단을 생성합니다.

- **난이도 가변 시스템**: `getColRange(level)` 함수를 통해 플레이어의 점수가 높아질수록 계단이 생성되는 좌우 범위를 동적으로 확장합니다. 이는 플레이어에게 더 넓은 시야와 빠른 판단력을 요구하여 자연스러운 난이도 상승을 유도합니다.
- **경로 최적화**: 이전 계단의 위치를 참조하여 플레이어가 이동 불가능한 경로가 생성되지 않도록 논리적으로 검증된 위치에만 블록을 배치합니다.

### 2. 무의식적 잔상 학습 엔진 (Subliminal Learning Engine)
플레이어의 몰입 상태를 저해하지 않으면서 교육적 효과를 얻을 수 있도록 **알파 블렌딩(Alpha Blending)** 기반의 잔상 시스템을 도입했습니다.

- **FloatingText 시스템**: 점프 및 특정 이벤트 발생 시, 현재 학습 중인 파이썬 코드 조각을 배경 레이어에 낮은 불투명도(0.05~0.1)로 렌더링합니다. 
- **인지 공학적 설계**: 명시적으로 공부를 강요하는 것이 아니라, 게임 플레이 중 시각적 노출을 반복함으로써 코딩 문법에 대한 친숙도를 높이는 무의식적 학습 기법을 적용했습니다.

### 3. 고성능 렌더링 파이프라인
안정적인 60FPS 환경을 위해 캔버스 렌더링 과정을 계층화하여 관리합니다.

- **레이어 구조**: 배경 잔상 레이어 -> 동적 계단 레이어 -> 캐릭터 및 이펙트 레이어 순으로 렌더링하여 시각적 깊이감을 제공합니다.
- **프레임 동기화**: `requestAnimationFrame`과 `deltaTime` 계산을 통해 기기 사양에 관계없이 일관된 물리 연산과 애니메이션 속도를 보장합니다.

## 📝 기술적 문제 해결 (Troubleshooting Case)

### ⚠️ 반응형 환경에서의 캔버스 좌표 동기화
- **문제**: 브라우저 화면 크기 변경 시 CSS 레이아웃 변화로 인해 마우스 클릭 위치와 실제 게임 블록의 좌표가 어긋나는 현상 발생.
- **해결**: `getBoundingClientRect()`를 활용해 렌더링된 캔버스의 실제 크기를 추적하고, 이를 물리 해상도와 매핑하는 보정 함수를 구현하여 모든 해상도에서 정확한 조작이 가능하도록 개선했습니다.

### ⚠️ 이미지 업로드 시 CORS 및 메모리 관리
- **문제**: 외부 URL로부터 이미지를 불러올 때 캔버스가 오염(Tainted)되어 데이터 추출이 불가능하거나 메모리 사용량이 누적되는 문제.
- **해결**: 이미지 객체에 `crossOrigin = "Anonymous"` 설정을 선행하고, 리사이징 완료 즉시 임시 객체를 명시적으로 해제하여 메모리 누수를 방지하고 보안 정책을 준수했습니다.
